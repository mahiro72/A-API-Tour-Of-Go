# 1 ~ 環境構築編 ~

この章では開発環境の構築、GoのAPIでHello,Worldを表示させるまでを目標とします。

## 技術選定

今回のAPIで使う技術は以下です。

```
- API
  - 言語: Go
  - ライブラリ:
    - HTTP: go-chi
    - DB:   gorm
- DB
  - MySQL
- Infrastructure
  - Docker, Docker Compose

```
